# LangGeAttribute - 饰品属性联动插件

## 项目简介
LangGeAttribute 是一个 Minecraft 1.12.2 Spigot 服务端插件，用于将 **LangGeShiPingAPI**（饰品系统）计算的属性数据自动同步到 **SX-Attribute**（属性系统），实现玩家佩戴饰品后获得对应战斗属性的功能。

## 核心功能
- ✅ **实时属性同步**：监听饰品更换事件，立即应用新的属性加成
- ✅ **自动加载机制**：玩家登录时自动读取并应用当前饰品属性
- ✅ **完整生命周期管理**：玩家退出时自动清除属性，防止数据残留
- ✅ **支持30+种属性**：包括攻击、防御、暴击、吸血、闪避、格挡、状态效果等

## 支持的属性列表
| 分类 | 属性 |
|------|------|
| 基础属性 | 攻击力、防御力、生命上限、移动速度 |
| 特殊攻击 | PVP/PVE伤害、破甲、吸血、生命恢复 |
| 暴击系统 | 暴击率、暴伤、暴抗 |
| 防御机制 | 闪避、格挡、反射、荆棘抵抗 |
| 状态效果 | 点燃、凋零、中毒、失明、缓慢、雷电、撕裂 |
| 增益属性 | 经验加成、掉落倍率、生命/防御百分比加成 |

## 前置依赖
- **必需**：[SX-Attribute](https://modrinth.com/plugin/sx-attribute) 2.0.2+
- **必需**：LangGeShiPingAPI 1.0-SNAPSHOT（需自行安装）

## 使用方法
1. 确保服务器已安装 SX-Attribute 和 LangGeShiPingAPI
2. 将 `LangGeAttribute.jar` 放入 `plugins` 文件夹
3. 重启服务器
4. 插件会自动加载默认配置（`config.yml`）

下载请到master分支 main分支并无东西
